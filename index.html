draggable Określa, czy element można przeciągać, czy nie
hidden Określa, że element nie jest jeszcze istotny lub przestał być istotny
id Określa unikalny identyfikator elementu
lang Określa język zawartości elementu
spellcheck Określa, czy element ma sprawdzać pisownię i gramatykę, czy nie
style Określa wbudowany styl CSS dla elementu
tabindex Określa kolejność tabulacji elementu
title Określa dodatkowe informacje o elemencie
translate Określa, czy treść elementu powinna zostać przetłumaczona, czy nie
